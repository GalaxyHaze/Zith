Getting Started

Installation

Install Zith on your system in just a few commands.

Prerequisites

  • Linux/macOS: Bash shell, curl, Git
  • Windows: PowerShell 5.0+, Git
  • Build Tools: C/C++ compiler (GCC, Clang, or MSVC)
  • Memory: At least 1 GB available

Linux & macOS

Use the official install script:

shell
curl -fsSL https://github.com/GalaxyHaze/Zith/raw/master/install.sh | bash

The script will:

  • Download the latest Zith release
  • Install to ~/.local/bin/zith
  • Add it to your PATH

Windows

Use PowerShell with the install script:

powershell
iex (New-Object Net.WebClient).DownloadString('https://github.com/GalaxyHaze/Zith/raw/master/install.ps1')

From Source

For developers wanting to build from source:

shell
git clone https://github.com/GalaxyHaze/Zith.git
cd Zith
mkdir build && cd build
cmake ..
make install

Verify Installation

Check that Zith is installed correctly:

shell
zith --version

You should see the version number and build information.

Troubleshooting

!
Command not found: Make sure ~/.local/bin is in your PATH. Add export PATH="$HOME/.local/bin:$PATH" to your ~/.bashrc or ~/.zshrc.
!
Permission denied: On Unix-like systems, run the install script with bash explicitly: bash install.sh
