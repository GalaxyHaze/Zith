Getting Started

Quick Start

Get up and running with Zith in just a few minutes.

1. Create a Project

shell
zith new hello-world

2. Write Your First Program

Navigate to your project and edit the main file:

zith
  import std.io.println as println;
  fn main(){
    println("Hello, Zith!");
  }

3. Build and Run

shell
zith build
zith run
