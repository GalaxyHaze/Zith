Project Configuration

ZithProject.toml

Configuration file for Zith projects. Defines metadata, dependencies, and build settings.

Overview

ZithProject.toml is a TOML configuration file that sits in the root of your project. It specifies project metadata, dependencies, and build configuration.

Basic Structure

toml
[project]
name = "my-app"
version = "0.1.0"
authors = ["Your Name <email@example.com>"]
description = "A Zith application"

[output]
bin = "bin/my-app"

[dependencies]
# Dependencies here

[dev-dependencies]
# Dev-only dependencies

[build]
# Build settings

[project] Section

Key Type Required Description
name string Yes Project name (alphanumeric + hyphens)
version string Yes Semantic version (e.g., 1.0.0)
authors array No Author names and emails
description string No Project description
license string No License (SPDX identifier)
repository string No Repository URL
keywords array No Search keywords

[output] Section

Specifies build output paths:

toml
[output]
bin = "bin/my-app"      # Binary output
lib = "lib/libmy.a"     # Static library
shared = "lib/libmy.so" # Shared library

[dependencies] Section

External crates your project depends on:

toml
[dependencies]
standard = "1.0"
math = { version = "2.1", features = ["simd"] }

[dev-dependencies] Section

Dependencies only for testing and development:

toml
[dev-dependencies]
test-framework = "0.5"

[build] Section

Build configuration options:

toml
[build]
opt-level = 3
debug = false
lto = true
panic = "abort"
Key Values Description
opt-level 0-3 Optimization level
debug true/false Include debug symbols
lto true/false Link-time optimization
panic unwind/abort Panic behavior

Example Complete File

toml
[project]
name = "web-server"
version = "1.2.3"
authors = ["Alice <alice@example.com>", "Bob <bob@example.com>"]
description = "A fast HTTP server"
license = "MIT"
keywords = ["http", "server", "networking"]

[output]
bin = "bin/web-server"

[dependencies]
http = "1.0"
socket = { version = "0.8", features = ["async"] }

[dev-dependencies]
test-client = "2.0"

[build]
opt-level = 3
debug = false
lto = true
ℹ
Run zith with no arguments and a ZithProject.toml present to automatically build the project.
