Raw Guide

Como usar (raw)

Estratégia prática para aplicar recursos avançados sem perder legibilidade.

Fluxo recomendado

  1. Comece com módulos claros e API pequena.
  2. Introduza traits quando houver comportamento compartilhado.
  3. Use generics para eliminar duplicação de tipos.
  4. Aplique macros/metaprogramação apenas em pontos repetitivos.

Checklist

  • Prefira clareza sobre “magia”.
  • Documente decisões avançadas no README do projeto.
  • Mantenha exemplos pequenos e compiláveis.
