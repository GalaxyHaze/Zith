Language Guide

Variables & Constants

Kalidous distinguishes between mutable variables and immutable constants at the syntax level.

Variable Declaration

Declare variables with the let keyword. By default, variables are immutable:

kalidous
let x = 5;           // Immutable
let mut y = 10;      // Mutable

Type Annotations

Explicitly specify types with a colon:

kalidous
let x: i32 = 42;
let mut name: str = "Alice";
let items: [i32] = [1, 2, 3];

Type Inference

Kalidous can infer types from context:

kalidous
let x = 42;      // Inferred as i32
let y = 3.14;    // Inferred as f64
let s = "hi";    // Inferred as str

Constants

Constants are compile-time values using the const keyword:

kalidous
const PI: f64 = 3.14159;
const MAX_SIZE: i32 = 1024;
Feature let let mut const
Mutable No Yes No
Type Required No (inferred) No (inferred) Yes
Runtime Yes Yes No

Shadowing

You can declare a new variable with the same name:

kalidous
let x = 5;
let x = x + 1;   // x is now 6
let x = "hello"; // x is now a string
ℹ
Prefer immutable bindings by default. Use mut only when necessary for clarity.
