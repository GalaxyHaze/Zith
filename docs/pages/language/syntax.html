Language Guide

Syntax Basics

Zith syntax is influenced by modern systems languages, favoring clarity and explicitness.

Hello World

zith
  import std.io.println as println;
  fn main() {
      println("Hello, World!");
  }

Comments

zith
// Single line comment

/* Multi-line
   comment */

/// Documentation comment
/// This appears in generated docs

Statements

Statements are instructions that end with a semicolon. Expressions return values.

zith
// Statement
let x = 5;

// Expression (no semicolon)
let y = {
  let x = 3;
  x + 1
};

Blocks

Blocks are enclosed in curly braces and return the value of the last expression:

zith
let result = {
  let a = 10;
  let b = 20;
  a + b  // Returns 30
};

Indentation & Formatting

Zith uses standard formatting conventions:

  • 2 or 4 spaces for indentation (no tabs)
  • Lines should be under 100 characters
  • Use zith fmt to auto-format code
