Language Guide

Functions

Functions are reusable blocks of code that can take parameters and return values.

Function Declaration

kalidous
fn greet() {
  println("Hello!");
}

greet();  // Call function

Parameters

Functions can accept typed parameters:

kalidous
fn add(a: i32, b: i32) {
  println(a + b);
}

add(5, 3);

Return Types

Specify what a function returns:

kalidous
fn add(a: i32, b: i32) -> i32 {
  a + b  // No semicolon = return value
}

let result = add(5, 3);

Early Returns

Use return to exit early:

kalidous
fn check(x: i32) -> str {
  if x < 0 {
    return "negative";
  }
  if x > 0 {
    return "positive";
  }
  "zero"
}

Function Pointers

Store and pass functions:

kalidous
fn apply(f: fn(i32) -> i32, x: i32) -> i32 {
  f(x)
}

Closures

Anonymous functions that capture variables:

kalidous
let y = 5;
let add_y = |x| x + y;
let result = add_y(3);  // 8

Documentation Comments

Document functions for users:

kalidous
/// Adds two numbers.
///
/// # Example
/// ```
/// let sum = add(2, 3);  // 5
/// ```
fn add(a: i32, b: i32) -> i32 {
  a + b
}

Visibility

Control function visibility with pub:

kalidous
pub fn public_fn() { }

fn private_fn() { }
ℹ
Functions should do one thing well. If a function is getting long, consider breaking it into smaller functions.
