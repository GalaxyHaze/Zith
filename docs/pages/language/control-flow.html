Language Guide

Control Flow

Control flow statements direct the execution of your code based on conditions and iterations.

if Expressions

Branching based on conditions:

zith
if x > 0 {
  println("positive");
} else if x < 0 {
  println("negative");
} else {
  println("zero");
}

if as Expression

Use if to assign values:

zith
let msg = if x > 0 { "pos" } else { "neg" };

while Loops

Repeat while a condition is true:

zith
let mut i = 0;
while i < 10 {
  println(i);
  i = i + 1;
}

for Loops

Iterate over collections:

zith
let arr = [1, 2, 3];
for item in arr {
  println(item);
}

Range Iteration

Iterate over numeric ranges:

zith
// Inclusive range
for i in 0..=10 {
  println(i);
}

// Exclusive range
for i in 0..10 {
  println(i);
}

break and continue

Control loop execution:

zith
for i in 0..100 {
  if i == 50 {
    break;  // Exit loop
  }
  if i % 2 == 0 {
    continue;  // Skip iteration
  }
  println(i);
}

match Expressions

Pattern matching on values:

zith
match value {
  0 => println("zero"),
  1 => println("one"),
  2..=9 => println("single digit"),
  _ => println("other"),
}

match with Enums

Safely handle different types:

zith
enum Option {
  Some(i32),
  None,
}

match opt {
  Option::Some(x) => println(x),
  Option::None => println("nothing"),
}
ℹ
Match arms must be exhaustive—all possible values must be covered.
