Language Guide

Types

Zith has a comprehensive type system with primitives, composites, and user-defined types.

Integer Types

Type Bits Range
i8 8 -128 to 127
i16 16 -32,768 to 32,767
i32 32 -2^31 to 2^31-1
i64 64 -2^63 to 2^63-1
u8 8 0 to 255
u16 16 0 to 65,535
u32 32 0 to 2^32-1
u64 64 0 to 2^64-1

Floating Point Types

Type Precision Example
f32 Single (32-bit) let x: f32 = 3.14;
f64 Double (64-bit) let x: f64 = 3.14159;

Other Primitives

zith
let is_active: bool = true;
let character: char = 'A';
let text: str = "Hello";

Array Types

Fixed-size arrays:

zith
let arr: [i32; 3] = [1, 2, 3];
let zeros: [i32; 5] = [0; 5];

Tuple Types

Heterogeneous fixed-size collections:

zith
let pair: (i32, str) = (42, "answer");
let x = pair.0;  // Access by index
let s = pair.1;

Struct Types

Define custom data structures:

zith
struct Point {
  x: i32,
  y: i32,
}

let p = Point { x: 10, y: 20 };

Enum Types

Define types with named variants:

zith
enum Color {
  Red,
  Green,
  Blue,
}

let c = Color::Red;

Type Aliases

Create alternative names for types:

zith
type UserId = u64;
let user: UserId = 12345;

Type Conversions

Convert between types using as:

zith
let x: i32 = 42;
let y: f64 = x as f64;  // Explicit conversion
let z: i64 = y as i64;
!

Zith requires explicit type conversions. Implicit casts are not allowed to prevent accidental data loss.

Pointer Types

Raw pointers for low-level memory access:

zith
let x: i32 = 42;
let ptr: *i32 = &x;  // Pointer to i32
