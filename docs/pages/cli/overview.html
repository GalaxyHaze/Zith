CLI Reference

CLI Reference

Complete reference for all Zith command-line tools and commands.

Available Commands

Command Description
check Syntax and type checking without compilation
compile Compile source files to machine code
build Build complete projects with dependencies
execute Run compiled binaries or bytecode
run Build and execute in one step
new Create projects or single .zith files
clean Remove build artifacts and cache files
repl Interactive shell for trying snippets
fmt Format source code with standard style rules
docs Generate API documentation

Global Options

Use these options with any command:

shell
-v, --verbose     Enable verbose output
-q, --quiet       Suppress non-essential output
-m, --mode MODE   Build mode: debug or release
💡
Run zith --help for a quick overview of all commands.
