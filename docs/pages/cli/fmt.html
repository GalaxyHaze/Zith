zith fmt

Format Zith source code using the standard style rules.

Synopsis

shell
zith fmt [path] [--check] [--write]

Description

Formats one file or all files in a project. Use --check in CI to fail when formatting is needed.

Examples

shell
# Format the current project
zith fmt

# Check formatting only
zith fmt --check

# Format one file
zith fmt src/main.zith
