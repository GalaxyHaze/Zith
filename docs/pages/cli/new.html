CLI Reference

zith new

Create a new project directory or a single Zith source file.

Synopsis

shell
zith new <name-or-path> [--bin] [--lib] [--force]

Description

zith new infers what to create from the argument you pass:

  • If you pass a project name such as my-project, it creates a project directory with starter files.
  • If you pass a filename ending in .zith such as main.zith, it creates a single source file.

Behavior

Input Result
zith new my-project Create project folder my-project/ with ZithProject.toml and src/main.zith
zith new main.zith Create file main.zith in current directory
zith new src/app.zith Create file and any missing parent directories

Options

Option Description
--bin Force binary project template when creating a project
--lib Force library project template when creating a project
--force Overwrite existing files

Examples

shell
# Create a new project
zith new my-project

# Create a single file
zith new main.zith

# Create a library project
zith new core-lib --lib
