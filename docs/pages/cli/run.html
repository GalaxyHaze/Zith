CLI Reference

zith run

Build and execute in a single command.

Synopsis

shell
zith run [target] [-m mode] [-- args]

Description

Convenience command that builds the project and immediately runs the resulting binary. Combines zith build and zith execute into one step.

Options

Option Description
-m, --mode MODE Build mode: debug or release
-- Separator for program arguments
-v, --verbose Verbose output

Examples

shell
# Build and run project
zith run

# Build release and run
zith run -m release

# Run with program arguments
zith run -- input.txt --output result.txt

# Debug build with verbosity
zith run -v

Behavior

  1. Checks if source files have changed
  2. If changed, rebuilds the project
  3. Executes the resulting binary
  4. Preserves program exit codes
ℹ
If the build fails, the program does not execute.

Performance Tip

For rapid iteration during development:

  • Use zith run for quick testing
  • Use zith build -m release before deployment
