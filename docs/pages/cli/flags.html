CLI Reference

Global Flags

Universal command-line options available with all Zith commands.

Help & Version

Flag Description Example
-h, --help Show help message zith build --help
--version Show version info zith --version

Output Control

Flag Description Example
-v, --verbose Show detailed output zith build -v
-q, --quiet Suppress non-error output zith build -q
-s, --strict Enable strict checks and warnings zith check --strict
--color auto|on|off Control colored output zith build --color off

Build Control

Flag Description Commands
-m, --mode MODE Build mode (debug/release) build, run
-j, --jobs N Parallel build jobs build
-O LEVEL Optimization (0-3) compile
-g Include debug symbols build, compile

Configuration

Flag Description
--config PATH Use alternate project config file
--env FILE Load environment variables from file

Common Patterns

Development Build (Fast)

shell
zith run -m debug

Release Build (Optimized)

shell
zith build -m release

Parallel Build

shell
zith build -j 8

Quiet Mode (CI/CD)

shell
zith build -q

Environment Variables

Variable Purpose
ZITH_MODE Default build mode
ZITH_JOBS Default parallel jobs
ZITH_HOME Config directory
