CLI Reference

zith repl

Start an interactive Read-Eval-Print Loop for quick experiments.

Synopsis

shell
zith repl [--load file] [--no-color]

Description

The REPL lets you evaluate expressions, inspect values, and prototype code without creating a project first.

Examples

shell
# Start interactive shell
zith repl

# Preload definitions
zith repl --load prelude.zith
ℹ
Use :help inside the REPL to list available meta-commands.
