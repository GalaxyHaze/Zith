CLI Reference

zith execute

Run compiled binaries or bytecode. Does not compile anything.

Synopsis

shell
zith execute [target] [--interpreted] [-v]

Description

Execute runs pre-compiled binaries or bytecode files. If the target is not found, it fails with an explicit error. This command is useful for running previously-built artifacts.

Target Resolution

When target is omitted, reads output from ZithProject.toml. If no toml exists, the command fails immediately.

Invocation Resolves to
zith execute toml.output
zith execute --interpreted toml.output.bc
zith execute bin/app bin/app

Options

Option Description
--interpreted Run bytecode with interpreter (not JIT)
-v, --verbose Show execution details
-- ARGS Pass arguments to the program

Examples

shell
# Run binary from toml
zith execute

# Run specific binary
zith execute bin/app

# Run with arguments
zith execute bin/app -- arg1 arg2

# Run bytecode with interpreter
zith execute program.bc --interpreted

Exit Codes

Exit codes from the executed program are preserved.

!
If the target binary is not found, execute exits with code 127.
