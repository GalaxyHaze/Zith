CLI Reference

zith clean

Remove build artifacts and cached outputs.

Synopsis

shell
zith clean [target] [--all] [--dry-run]

Description

zith clean deletes generated artifacts such as binaries, bytecode, and intermediate cache files.

Options

OptionDescription
--allAlso remove shared caches and downloaded dependencies
--dry-runShow what would be removed without deleting files
-v, --verboseDisplay each removed path

Examples

shell
# Remove local build output
zith clean

# Preview removals
zith clean --dry-run

# Full cleanup
zith clean --all
