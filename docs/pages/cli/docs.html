CLI Reference

zith docs

Generate API documentation from your project source.

Synopsis

shell
zith docs [--open] [--output dir] [--private]

Description

Builds HTML docs from module comments, public items, and type signatures. Generated output can be published or viewed locally.

Examples

shell
# Generate docs into default directory
zith docs

# Generate and open in browser
zith docs --open

# Include private items
zith docs --private
